3. 自动装箱与拆箱

    装箱：基本数据类型转为封装类型
    
    拆箱：封装类型转为基本数据类型

> 拆箱
```
    Integer i2=Integer.valueOf(10);
    int i3=i2;
```
> 装箱
```
int i4=8;
Integer i5=i4;
```
